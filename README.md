# ИИ Система Генерации Уровней на основе Фотографий

Полнофункциональная система для автоматической генерации игровых уровней из изображений с использованием компьютерного зрения и обработки изображений.

## Возможности

✨ **Основной функционал:**
- Анализ изображений и автоматическое обнаружение структур для создания уровней
- Классификация типов платформ (нормальные, опасные, собираемые, движущиеся)
- Генерация JSON данных уровня совместимого с Unity
- REST API для обработки изображений
- Консольное приложение для локальной генерации
- Полная интеграция с Unity

## Классы и их назначение

### Основные классы обработки

- **ImageProcessor** - Обработка изображений (преобразование в оттенки серого, обнаружение краёв)
- **LevelGenerator** - Генерация уровня и сохранение в JSON
- **AdvancedImageAnalyzer** - Продвинутый анализ с классификацией типов платформ

### Unity интеграция

- **LevelLoaderUnity** - Загрузка JSON данных в Unity
- **PlatformBuilder** - Создание визуальных платформ
- **LevelManager** - Управление жизненным циклом уровня

### API

- **LevelGenerationController** - REST endpoint для обработки изображений
  - `POST /api/levelgeneration/generate` - Загрузка файла
  - `POST /api/levelgeneration/generate-from-url` - Обработка по URL
  - `GET /api/levelgeneration/health` - Проверка статуса

## Установка

### Требования
- .NET 8.0 SDK
- Visual Studio / VS Code (опционально)

### Шаги установки

1. Клонируйте репозиторий
2. Откройте папку проекта
3. Установите зависимости:
```bash
dotnet restore
```

## Использование

### 1. Консольное приложение

Для локальной генерации уровня:

```bash
cd c:\Диплом
dotnet run ConsoleLevelGenerator.cs
```

Результаты:
- `input.png` - Тестовое входное изображение
- `output.png` - Обработанное изображение с обнаруженными краями
- `level.json` - Данные сгенерированного уровня

### 2. REST API

Запуск веб-сервиса:

```bash
dotnet run
```

Затем используйте API endpoints:

#### Загрузка файла
```bash
curl -X POST -F "imageFile=@image.png" https://localhost:5001/api/levelgeneration/generate
```

#### Обработка по URL
```bash
curl -X POST -H "Content-Type: application/json" \
  -d '{"imageUrl":"https://example.com/image.png"}' \
  https://localhost:5001/api/levelgeneration/generate-from-url
```

#### Проверка статуса
```bash
curl https://localhost:5001/api/levelgeneration/health
```

### 3. Unity интеграция

1. Импортируйте скрипты в ваш Unity проект:
   - `LevelLoaderUnity.cs`
   - `PlatformBuilder.cs`
   - `LevelManager.cs`

2. Создайте GameObject с компонентом `LevelManager`

3. Назначьте параметры:
   - Platform Prefab - префаб платформы
   - Platform Container - родительский контейнер для платформ

4. При запуске сцены уровень будет автоматически загружен

```json
{
  "Width": 600,
  "Height": 400,
  "Platforms": [
    {
      "Start": {"X": 50, "Y": 200},
      "End": {"X": 250, "Y": 200},
      "Height": 20
    },
    ...
  ]
}
```

## Типы платформ

- **Normal** (0) - Черные платформы - обычные платформы для прохождения
- **Dangerous** (1) - Красные платформы - враждебные платформы, повреждают игрока
- **Collectible** (2) - Желтые платформы - собираемые бонусы
- **Moving** (3) - Синие платформы - движущиеся платформы

## Примеры использования

### Пример 1: Простая генерация в консоли

```csharp
var analyzer = new AdvancedImageAnalyzer();
var levelData = analyzer.AnalyzeWithAdvancedVision("my_image.png");

var generator = new LevelGenerator();
generator.SaveLevelToJson(levelData, "output.json");
```

### Пример 2: Обработка с целью отладки

```csharp
var processor = new ImageProcessor();
processor.ProcessImage("input.png", "processed.png");

// Теперь посмотрите processed.png для проверки обнаружения краев
```

## Оптимизация производительности

- **MIN_PLATFORM_LENGTH** - Минимальная длина платформы (по умолчанию 20 пиксель)
- **VERTICAL_MERGE_THRESHOLD** - Расстояние для объединения платформ (по умолчанию 3 пиксель)

Отрегулируйте эти параметры в `AdvancedImageAnalyzer` для входных изображений разных размеров.
