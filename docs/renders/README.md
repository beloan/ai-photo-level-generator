# Рендеры рельефа: до / после подключения ИИ-модели глубины

Сравнение строится на одном и том же фото. Камера и освещение одинаковы — отличается только
**источник высоты рельефа**:

- **до** — высота из яркости пикселей фото (классический метод);
- **после** — высота из **карты глубины нейросети Depth Anything V2** (`analysis_source = depth-anything-v2`).

## Пример 1 (рекомендованный) — горный пейзаж
Исходное фото: `samples/demo_highlands.jpg` (зелёный горный хребет с дорогой, явная глубина
near→far). Такой кадр модель распознаёт легко: гладкие крупные формы, понятный передний/задний план.

| Файл | Что показывает |
|------|----------------|
| `demo_highlands_before.jpg` | Рельеф из яркости — «комковатый», не совпадает с реальной формой склона. |
| `demo_highlands_after.jpg` | Рельеф из глубины модели — цельная зелёная гора по реальной геометрии сцены. |

## Пример 2 — зимний пейзаж (наглядный провал классического метода)
Исходное фото: `wwwroot/uploads/input.jpg` (дом, голые деревья, снег, светлое небо).

| Файл | Что показывает |
|------|----------------|
| `relief_before_brightness.jpg` | Яркость: тёмные ветки дают рваные «иглы», яркий снег/небо — плоские. |
| `relief_after_depth.jpg` | Глубина модели: гладкая поверхность по форме сцены, без шумовых артефактов. |

## Какие фото подходят лучше всего
- ✅ Гладкие крупные формы с явной глубиной: горы/холмы, дюны, дорога/поле, уходящие к горизонту.
- ⚠️ Хуже: много тонкой высокочастотной мелочи (голые ветки), большие плоские области неба/воды,
  люди/предметы крупным планом.

## Как получено
1. `ai_service/` (FastAPI + Depth Anything V2 Small) по фото отдаёт карту глубины.
2. Бэкенд `/api/generate` берёт реальную глубину и кладёт её в анализ сцены.
3. `wwwroot/viewer-3d.js` строит рельеф из глубины (билинейная выборка + сглаживание),
   фото драпируется текстурой. Переключение источника — `Viewer3D.setReliefMode('depth'|'photo')`.

Тех. вывод для работы: яркость ≠ форма сцены, поэтому классический метод шумит; монокулярная
нейросетевая оценка глубины даёт корректный и гладкий рельеф из обычной фотографии.
